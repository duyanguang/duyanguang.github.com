解这道题的思路要想像我们在盖房子，这个房子有几个单元，就是列表的长度。但是这个楼有一个特点，每个单元楼的高度是不一样的。最高的单元楼就是列表中最大的那个元素，然后我们就从最高那层楼逐层开始盖房子（向下遍历），跟我们现实中盖房子不太一样。如果我这个单元楼的高度比当前楼层高，我就盖一层（打*号），否则我就不盖（打空格），一直盖到一层为止。
